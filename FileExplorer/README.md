# File Explorer

A simple file explorer application built using Python and Tkinter.

![File Explorer Screenshot](https://i.imgur.com/VlcLVzz.png)

## Features

- Browse and navigate through directories on your system.
- View the list of files and folders in the selected directory.
- Open and view **text files** within the application (limited to text file formats).
- User-friendly graphical interface.

## Usage

1. Run the `FileExplorer.py` script using Python.
2. The main window of the File Explorer will appear.
3. Use the "Browse" button to select a directory you want to explore.
4. The list of files and folders in the selected directory will be displayed.
5. Select a text file and click "Open File" to view its content within the application.
6. The content of the file will be displayed in a new window.

## Dependencies

- Python 3.x
- Tkinter (usually included with Python)

## Notes

- This program is a basic file explorer and text file viewer.
- It is designed to provide a simple way to navigate directories and view text files.
- The program is limited to opening **text files** only and may not support other file types.
- Feel free to customize and enhance the program based on your needs.
- The program may have limitations and may not handle all types of files.
